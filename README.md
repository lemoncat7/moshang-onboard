# MoShang Onboard

莫殇的状态看板 Dashboard

## 功能

- ⚡ 实时状态显示（心跳、任务、消息）
- 🎯 中心环绕浮窗布局
- 📊 数据自动更新

## 启动

```bash
node scripts/server.js
```

## 访问

- 看板: http://localhost:19000
- Star Office UI: http://localhost:19000/star

## 技术栈

- Node.js HTTP Server
- Vanilla JavaScript
- CSS3 动画
